# 簡単PDFカレンダー デザインについて

## フォルダ構成

```
/designs
  +-- {デザインファイルフォルダ} ※任意の名前
        +-- info.json ※デザインの名称や設定など
        +-- main.svg ※カレンダーのデザイン、info.jsonで名称を指定
```

## カレンダーのデザインSVGの仕様

Inkscape固有の属性名を利用するため、基本的には Inkscape を利用しデザインします。

SVGで表示されている項目は全て表示されます。

### 日付変数

`rect` 要素を期待しています。
また、 `inkscape:label` が下記の表の仕様に沿っていることを期待しています。

|`inkscape:label`<br/>の正規表現|説明|
|:-|:-|
|`^year\[.+\]`|年を矩形内に描画します|
|`^month\[.+\]`|年を矩形内に描画します|

|対象|指示|説明|
|-|-|-|
|`year` |`short`|短い表記。`2025`など|
|`year` |`long`|長い表記。`2025年`など|
|`year` |`jaJP`|結果を日本語で返します|
|`month` |`short`|短い表記。`３月`や`Feb`など|
|`month` |`long`|長い表記。`弥生`や`February`など|
|`month` |`jaJP`|結果を日本語で返します|
|`month` |`enJP`|結果を英語で返します|
