# オンラインカレンダーメーカー

画像を含んだカレンダーを作りPDFで書き出せるウェブアプリ。

# できること

* 複数のカレンダーのデザインを選択
* ファイル選択かファイルドロップで簡単に画像を指定
* **日本の** 祝日をカレンダーに反映する
* カレンダーをPDFとして出力できる

# 使い方


# 開発

開発用サーバーの起動

```console
$ npm ci
$ npm run dev
```

リリースビルド

```console
$ npm run build
```
